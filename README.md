## Installation

Install with npm

```bash
  npm install
```

To start the server, start a docker postgres database ( don't forget to correctly fill the env file and change the docker-compose.yml file to fit your need, ex : change the volume location )

```bash
  docker compose up -d
  npm run dev
```
